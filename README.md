## 📌 About This Repository  

This repository contains the documentation and analysis performed for **Task 2 – Phishing Email Detection & Awareness System** as part of the Cyber Security Internship program.

The objective of this task is to analyze phishing emails, identify malicious indicators, classify email types, and create awareness documentation explaining phishing risks and prevention techniques.

All analysis was performed in a controlled environment using publicly available phishing samples.

---

## 🎯 Task Objective  

- Identify phishing indicators (spoofed sender, malicious links, fake domains)
- Classify emails as **Safe / Suspicious / Phishing**
- Perform URL safety verification
- Conduct email header and link analysis
- Provide awareness and prevention guidelines

---

## 🛠 Tools & Platforms Used  

- Google Safe Browsing  
- VirusTotal  
- Browser Developer Tools  
- Email Header Analysis  
- Public Phishing Sample Datasets  
- Git & GitHub  

---

## 🔍 Methodology  

1. Collected phishing and legitimate email samples.
2. Analyzed sender details and domain authenticity.
3. Examined embedded links and checked URL safety using VirusTotal and Google Safe Browsing.
4. Identified phishing indicators such as:
   - Urgency language
   - Fake branding
   - Suspicious domains
   - Misleading hyperlinks
5. Classified emails based on risk level.
6. Documented findings and created awareness guidelines.
